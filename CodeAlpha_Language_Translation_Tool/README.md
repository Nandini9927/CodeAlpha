# AI Language Translation Tool

## About the Project
AI Language Translation Tool is a web-based application that enables users to translate text between multiple languages instantly. It provides a simple and interactive interface for real-time language conversion, helping users communicate across different languages with ease.

Users can:
- Translate text between multiple languages
- Select source and target languages
- Swap languages instantly
- Copy translated text easily

## Features
- Real-time text translation
- Multiple language support
- Clean and responsive UI
- Language swap functionality
- Fast and accurate translations
- User-friendly experience

## Tech Stack
- React.js
- JavaScript
- HTML
- CSS
- Vite
- Translation API
